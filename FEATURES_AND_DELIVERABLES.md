## 🔄 API Endpoints

### Authentication
- POST `/auth/register` - Create account
- POST `/auth/login` - Sign in
- GET `/auth/me` - Get current user
- PUT `/auth/update-profile` - Update profile
- PUT `/auth/change-password` - Change password
- POST `/auth/logout` - Sign out

### Conversions
- POST `/convert/merge-pdf` - Merge PDFs
- POST `/convert/split-pdf` - Split PDF
- POST `/convert/compress-pdf` - Compress
- POST `/convert/rotate-pdf` - Rotate
- POST `/convert/pdf-to-jpg` - PDF to image
- POST `/convert/jpg-to-pdf` - Image to PDF
- POST `/convert/pdf-to-word` - PDF to Word (Premium)
- POST `/convert/watermark-pdf` - Add watermark (Premium)
- POST `/convert/protect-pdf` - Add password (Premium)
- GET `/convert/history` - Conversion history
- POST `/convert/pdf-info` - Get PDF info

### Payments
- POST `/payment/create-checkout` - Create session
- POST `/payment/create-portal` - Billing portal
- GET `/payment/subscription` - Get subscription
- POST `/payment/cancel` - Cancel subscription
- GET `/payment/plans` - Available plans
- POST `/payment/webhook` - Stripe webhook
